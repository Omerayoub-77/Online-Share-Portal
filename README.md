# 🌐 Online Share Portal

A modern web-based **Online Share Portal** developed to simplify secure file sharing, document management, and online collaboration. The system allows users to upload, manage, and share files efficiently through an easy-to-use interface.

---

# 🚀 Features

## 👤 User Module

* User Registration & Login
* Secure Authentication
* Upload & Share Files
* Download Shared Files
* Manage Uploaded Documents
* User Dashboard

## 📂 File Management

* Upload Multiple File Types
* Organize Shared Content
* Delete & Update Files
* File Access Management
* Storage Monitoring

## 🔒 Security Features

* Secure Login System
* Session Management
* Access Control
* File Validation
* Protected File Sharing

## 🛠️ Admin Module

* Manage Users
* Monitor Uploaded Files
* System Administration
* User Access Control
* Portal Monitoring

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Online-Share-Portal.git
cd Online-Share-Portal
```

---

## 2️⃣ Configure Database

1. Open phpMyAdmin / SQL Server
2. Create a database
3. Import the provided SQL file

---

## 3️⃣ Configure Local Server

Move the project folder to:

```bash
htdocs/
```

Start:

* Apache
* MySQL

---

## 4️⃣ Run the Application

Open browser:

```bash
http://localhost/Online-Share-Portal
```
